# 高仿饿了吗APP

> 项目框架: vue2/vue-router2/vue-resourse

> 构建工具: vue-cli(webpack)

> 预处理器: sass(scss)

> 语法检查: es-lint

> 布局方案: flex

> 包管理工具: yarn(很快，我hold不住)

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
